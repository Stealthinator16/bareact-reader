# Bareact Reader

Digital reader for Indian statutes. Presents legislative text in an editorial, book-like layout with margin annotations for commentary, case law references, and technical deep dives.

## Features

- Navigate statutes by section with a library view and per-section reader
- Editorial typography using Crimson Pro for body text and Inter for UI elements
- Margin annotation layout with sidenotes for commentary, case law, and technical details
- Responsive design that collapses gracefully on smaller screens
- Search across statute content

## Included Statutes

- Digital Personal Data Protection Act, 2023 (DPDP Act)
- Information Technology Act, 2000

## Tech Stack

- Vite
- Vanilla JavaScript (ES modules)
- CSS (custom properties, grid, flexbox)

## Setup

```
npm install
npm run dev
```

To build for production:

```
npm run build
npm run preview
```
